# MCP Server for httpstat

MCP server for httpstat

Generated by [mcp-anything](https://github.com/gabrielekarra/mcp-anything).

## Installation

```bash
pip install -e .
```

### Target application dependencies

This server calls into the target application. Install its dependencies:

```bash
pip install -e /tmp/httpstat
```

## Usage

### As an MCP server (stdio transport)

```bash
mcp-httpstat
```

### Claude Desktop configuration

Add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "httpstat": {
      "command": "mcp-httpstat"
    }
  }
}
```

## Available Tools

### `run_httpstat`

Run httpstat with the given command-line arguments

**Parameters:**
- `args` (string): Command-line arguments to pass to httpstat (e.g. a URL or flags)
