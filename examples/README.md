# Examples

Real-world MCP servers generated by `mcp-anything`. Each example was produced with a single command, no manual editing.

## httpstat

**Source:** [reorx/httpstat](https://github.com/reorx/httpstat) — a Python CLI tool that visualizes curl timing statistics.

**Command:**
```bash
mcp-anything generate /path/to/httpstat --name httpstat --no-llm
```

**What was detected:** CLI application with argparse (2 source files, confidence 0.60)

**Generated tools (2):**
| Tool | Description |
|------|-------------|
| `run_httpstat` | Run httpstat with command-line arguments (e.g. a URL or flags) |
| `print_help` | Print help |

**Generated resources:** status, commands, tool-index

**mcp.json config:**
```json
{
  "mcpServers": {
    "httpstat": {
      "command": "mcp-httpstat",
      "args": []
    }
  }
}
```

**Output:** [`httpstat-server/`](httpstat-server/)

## click

**Source:** [pallets/click](https://github.com/pallets/click) — the Python package for creating command-line interfaces, used by Flask, Celery, and thousands of other projects.

**Command:**
```bash
mcp-anything generate /path/to/click --name click --no-llm
```

**What was detected:** CLI framework (confidence 0.90, 7 signals) + Python API (confidence 0.80, 45 signals). 18 source files scanned, 129 functions and 71 classes found.

**Generated tools (50):** Full tool list including `command_make_context`, `command_get_help`, `choice_convert`, `confirm`, `clear`, `cli_runner_isolated_filesystem`, and 44 more. Tools are auto-grouped by category (argument, command, choice, cli_runner, etc.).

**Generated resources:** status, commands, tool-index, plus per-category documentation resources

**mcp.json config:**
```json
{
  "mcpServers": {
    "click": {
      "command": "mcp-click",
      "args": []
    }
  }
}
```

**Output:** [`click-server/`](click-server/)

## Generating your own

```bash
pip install mcp-anything
mcp-anything generate /path/to/any/project --name my-project
```

The generated server will be in `mcp-my-project-server/`. Install it with `pip install -e mcp-my-project-server/` and add the mcp.json config to your MCP client.
